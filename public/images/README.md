# BBM Images

This directory contains images for the Brampton Beit Midrash website.

## Required Images

### Logo
- `bbm-logo.png` - Brampton Beit Midrash logo

### Rabbis
Place the following rabbi photos in the `rabbis/` subdirectory:

- `rabbi-kahlani-portrait.png` - Rabbi Natan Kahlani (main portrait for leadership section)
- `rabbi-kahlani-profile.png` - Rabbi Natan Kahlani (alternative photo, if needed)
- `rabbi-hecht.png` - Rabbi Hecht
- `rabbi-bazak.png` - Rabbi Bazak
- `rabbi-hye.png` - Rabbi Hye
- `rabbi-goldstien.png` - Rabbi Goldstien

## Notes

- Images should be optimized for web (PNG or JPG format)
- Recommended dimensions: 
  - Logo: 400x400px or larger
  - Rabbi portraits: 800x800px or larger (square aspect ratio)
- Next.js will automatically optimize images at build time
